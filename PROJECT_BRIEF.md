# Keepnoto Project Brief

Keepnoto is a modern link-saving and personal knowledge organization app.

## Problem

People save links, articles, videos, tools, references, and ideas, but later forget why they saved them. Over time, saved links become a messy graveyard. Users either never return to them or delete everything because the context is gone.

## Value

Keepnoto helps users save links with clear context, especially the reason why something was saved. It should make saved links easy to understand, organize, search, and return to later.

## Core Idea

Keepnoto is not just a bookmark manager. It is a personal knowledge vault for saved links, references, ideas, and useful materials.

## Primary User

Anyone who saves links and wants to return to them later with context, including designers, developers, students, creators, researchers, founders, marketers, and people collecting online materials.

## Product Focus

- Capture the reason a link was saved.
- Make saved links feel useful, valuable, and revisitable.
- Reduce link chaos without adding too much organization work.
- Keep saving simple and returning later easy.
- Avoid feature creep that distracts from contextual link saving.
